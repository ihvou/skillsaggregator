# Demo Script

1. Open the public web root and show the badminton skill grid.
2. Open `/badminton/forehand-smash` and show SEO content, resource groups, related skills, and JSON-LD in page source.
3. Open `/admin`, choose a skill, and run Link Searcher.
4. Show the moderation queue receiving a pending suggestion with model votes.
5. Approve the suggestion and refresh the skill page.
6. Open the Expo app, tap Badminton, tap the same skill, filter by level, save a resource, mark it completed, and open the Saved tab.
7. Open `/admin/runs` and show run history with suggestion and triangulation counts.
